# Berthoud WiFi

Static marketing website for **Berthoud WiFi**, a Berthoud-based UniFi installation and support company serving Northern Colorado.

## Deployment

This repository is designed for Cloudflare Pages.

- Production branch: `main`
- Framework preset: `None`
- Build command: leave blank
- Build output directory: `/`
- Custom domain: `berthoudwifi.com`

## Current structure

- `index.html` — homepage and local business structured data
- `styles.css` — responsive site design
- `script.js` — mobile navigation and footer year
- `robots.txt` — crawler instructions
- `sitemap.xml` — initial sitemap
- `_headers` — Cloudflare Pages security and caching headers
- `assets/berthoud-wifi-logo.png` — primary PNG logo
